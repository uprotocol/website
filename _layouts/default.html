Where is the specification?

Although the JVM implementation of uprotocol is relatively stable, there hasn't been an initiative to document the way the protocol works, as that would take a lot of time.

If you care about this happening, please give the specs repo a star on GitHub so that we can know you are interested. You can also approach us so that we can talk about how you can help.
